웹페이지 html구조 보는 방법

1. 개발자 도구에 들어간다.

2. console 을 누른다.

3. ['div', 'span', 'ul', 'li', 'dd', 'dl', 'section', 'h1', 'a', 'img', 'form', 'button', 'header', 'footer', 'input', 'p'].forEach(e => { document.querySelectorAll(e).forEach(element => { element.style.outline = "1px solid dodgerblue" }) }) 를 붙여넣기 한다.
